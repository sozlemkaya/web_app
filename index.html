Explore the foundational services of Amazon Web Services. Click on any service to learn more!

EC2 (Elastic Compute Cloud)

Virtual servers in the cloud. Resizable compute capacity in minutes.

What it is: EC2 provides configurable virtual machines (instances) that you can rent to run your applications.

Use Cases: Hosting web servers, running batch processing, application hosting, and more.

Key Features: Various instance types, scaling options, different purchasing models (On-Demand, Reserved, Spot).

S3 (Simple Storage Service)

Scalable object storage for the internet. Highly durable and available.

What it is: S3 is an object storage service that offers industry-leading scalability, data availability, security, and performance.

Use Cases: Data lakes, backup and restore, archiving, disaster recovery, content distribution.

Key Features: Buckets, objects, versioning, lifecycle policies, encryption.

Lambda

Run code without provisioning or managing servers. Serverless compute.

What it is: Lambda lets you run code for virtually any type of application or backend service without any administration.

Use Cases: Event-driven processing, backend for mobile/web apps, data processing, chatbots.

Key Features: Pay-per-use, automatic scaling, integrates with many other AWS services.

VPC (Virtual Private Cloud)

Logically isolated section of the AWS Cloud to launch resources.

What it is: VPC enables you to provision a logically isolated section of the AWS Cloud where you can launch AWS resources in a virtual network that you define.

Use Cases: Network isolation, hybrid cloud setups, secure environments for sensitive data.

Key Features: Subnets, routing tables, network ACLs, security groups, internet gateways.

RDS (Relational Database Service)

Managed relational databases in the cloud. Simplifies database setup.

What it is: RDS makes it easy to set up, operate, and scale a relational database in the cloud. It provides cost-efficient and resizable capacity while automating time-consuming administration tasks.

Use Cases: Web and mobile applications, enterprise applications, any application requiring a relational database.

Key Features: Supports multiple database engines (MySQL, PostgreSQL, Oracle, SQL Server, MariaDB, Aurora), automated backups, multi-AZ deployments.

DynamoDB

Fast and flexible NoSQL database service for all applications.

What it is: DynamoDB is a fully managed, serverless, key-value NoSQL database designed to run high-performance applications at any scale.

Use Cases: Mobile, web, gaming, ad tech, IoT, and other applications that need low-latency data access.

Key Features: High performance at scale, fully managed, global tables, on-demand capacity.
